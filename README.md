**PlayBook** is a React web app to manage CS2 tactics. Keep track of strategies by map, side, zone, description, and effectiveness.

## Features

- Add, edit, and delete tactics
- View tactic details in a modal
- Search tactics by map
- Responsive design with DaisyUI/Tailwind
- Toast notifications for actions

## Tech

- React (useState, useEffect)
- DaisyUI / Tailwind CSS
- Reusable components (TableList, ModalForm, Toast)

## Setup

1. Clone the repo
2. Run `npm install`
3. Run `npm start`
